Tu stack técnico en contexto global · 2026
▶ clic en cada nodo para expandir
Captura móvil → AUM▶
AUM (iOS)
Host de audio modular en móvil. Captura de ideas en campo, prototipos rápidos. Primera etapa del flujo creativo.
Flujo móvil → DAW
Ideas en AUM → exportar audio/MIDI → continuar en Ableton. Ciclo de ideación ágil sin depender del computador.
DAW principal → Ableton▶
Ableton Live
Centro de producción, grabación y arranging. Session View para live performance. Arrangement para composición.
Ableton + IA (tendencia)
Ableton 12 incorpora transformación de audio con ML. Integración creciente con herramientas generativas.
Síntesis → Pd + Max for Live▶
Pure Data
Síntesis en tiempo real, procesamiento de señal, generación algorítmica. Patches personalizados como instrumentos únicos.
Max for Live
Integración de Pd/Max dentro de Ableton. Instrumentos y efectos personalizados. ML disponible (ml.star, etc).
Tendencia emergente
Síntesis neural, análisis espectral IA y generación de gestos sonoros dentro de patches Pd/Max.
Control → TouchOSC + Protokol▶
TouchOSC
Superficie de control OSC/MIDI personalizable desde móvil/tablet. Diseño de interfaces propias para live set.
Protokol
Monitoreo y mapeo MIDI. Visualización de datos de controladores en tiempo real.
Hardware físico
Controladores MIDI, pedales, guitarras. Cadena analógica → digital → procesamiento → output.
Instalación sonora▶
Tu stack aplicado
Pd corriendo patches autónomos, TouchOSC como interfaz de control espacial, sensores → OSC → síntesis.
Ampliaciones posibles
libpd para instalaciones web, Raspberry Pi + Pd para hardware autónomo, integración de IA generativa.
Apps + gestión (stack profesional)▶
Sistema Operativo GEF
Python + Google Sheets + Notion + Telegram + Netlify + GitHub. Activo en producción.
Convergencia
El mismo stack puede gestionar residencias, lanzamientos musicales y convocatorias — no solo el equipo.
